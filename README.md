# 🛡️ TrustLens AI
### AI-Powered Misinformation Detection & News Verification Platform

TrustLens AI is an AI-powered web application designed to detect misinformation and verify the credibility of news articles. The platform analyzes news content, identifies suspicious patterns, assigns a confidence score, and provides users with an easy-to-understand credibility assessment.

---

## 🚀 Features

- 🔍 AI-powered news verification
- 📊 Confidence score with visual progress bar
- 📜 Verification history
- 🖼️ News screenshot upload
- 🤖 AI analysis and recommendation
- ⚡ Fast backend API using Express.js
- 💾 Local storage for history
- 📱 Responsive and modern UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/vishal-r-cse/trustlens-ai.git
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

---

### Backend Setup

```bash
cd backend
npm install
node server.js
```

Backend runs at:

```
http://localhost:5000
```

---

## 🔍 How It Works

1. User enters a news article or headline.
2. User can optionally upload a screenshot.
3. Frontend sends the news to the backend API.
4. Backend analyzes suspicious keywords and misinformation indicators.
5. TrustLens AI generates:
   - Fake/Genuine prediction
   - Confidence score
   - AI analysis
   - Recommendation
6. Verification history is saved locally.
